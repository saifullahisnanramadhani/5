Kesimpulan skenario 1 : Dengan menggunakan grid, item-item yang ada pada tempat_grid bisa diatur menjadi lebih fleksibel.

Kesimpulan skenario 2 : Kode program tersebut adalah contoh penggunaan CSS Grid untuk membuat tata letak grid sederhana.

Kesimpulan skenario 3 : Grid gap untuk mengatur jarak setiap item. Gap column berfungsi untuk mengatur jarak pada column, dan gap row berfungsi mengatur jarak antar row.

Kesimpulan skenario 4 : Column gap dan row gap dapat diatur sekaligus dengan grid-gap.
